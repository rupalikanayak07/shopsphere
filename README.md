# ShopSphere 🛒

A full-stack Django-based eCommerce web application.

## Features
- Product listing
- Category filtering
- Add to cart
- Trending & offer products
- Beautiful UI

## Tech Stack
- Django
- HTML, CSS
- SQLite (development)

## Setup

```bash
git clone <https://github.com/rupalikanayak07/shopsphere/>
cd myproject
pip install -r requirements.txt
python manage.py runserver
